プライバシーポリシー（個人情報保護方針）

対象アプリ：Hidden Octopus

運営者（パブリッシャー）：OwnReview

制定：2026年6月23日

本アプリは、利用者の情報について以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を 定めます。本ポリシーは、本アプリがどのような情報を取得し、それをどのように利用・共有するか、利用者が どのようにご自身の情報を管理できるかをご説明するものです。

重要な宣言
当方は、利用者の個人情報を第三者に販売いたしません。本アプリは全年齢を対象としていますが、13歳未満の 利用者を主な対象とはしていません。利用者の皆様に安心してご利用いただけるよう、プライバシーの保護を 最優先に取り組んでいます。

1．お問い合わせ窓口

本アプリの情報の取扱いに関するお問い合わせは、以下までご連絡ください。

  • 運営者：OwnReview
  • 連絡先：contact@ownreview.jp

2．取得する情報

本アプリは、利用者の氏名・メールアドレス・電話番号・生年月日などの個人を直接特定する情報を取得しません。 本アプリにはアカウント登録機能・投稿機能はありません。本アプリが取り扱う情報は以下に限られます。

⑴ 広告配信に伴い自動的に取得される情報（Google AdMob）

  • デバイス識別情報（広告ID：iOS の IDFA / Android の AAID）
  • 端末情報（OS、機種、言語設定等）
  • 大まかな位置情報（IPアドレスに基づく国・地域レベル）
  • 広告の表示・操作に関する情報

⑵ オンライン対戦機能に伴う情報（Unity Gaming Services）

  • 匿名の認証ID（氏名・メール等と紐づかない、サービスが自動発行する匿名識別子）
  • マルチプレイ接続のための一時的な接続情報（対戦セッション中のみ利用）

3．情報の利用目的

当方は、取得した情報を以下の目的でのみ利用します。

  • ⑴ 本アプリ内での広告の表示・配信のため
  • ⑵ 広告効果の測定のため
  • ⑶ オンラインマルチプレイ機能の提供のため
  • ⑷ 本アプリの不具合の調査・品質改善のため
  • ⑸ 本アプリの安全性確保・不正利用防止のため

4．第三者への情報提供

当方は、法令で認められた場合を除き、利用者の同意なく情報を第三者に販売または提供することはありません。 本アプリは、サービス提供のため以下の第三者サービスを利用しており、上記の情報がこれらに送信されます。

  • Google AdMob（広告配信）
    • AdMob／広告に関するプライバシー：https://policies.google.com/technologies/ads
    • Google プライバシーポリシー：https://policies.google.com/privacy
  • Unity Gaming Services（マルチプレイ）
    • Unity プライバシーポリシー：https://unity.com/legal/privacy-policy

5．国際データ移転について

本アプリが取得した情報は、サービス提供のため Google 社・Unity 社など海外企業のサーバーに保存・処理される 場合があります。これらの企業は、適切なデータ保護基準を遵守しています。

6．広告とトラッキングについて

  • 本アプリは、利用者の興味に基づくパーソナライズド広告を表示する場合があります。
  • iOS をご利用の場合：アプリ起動時に「トラッキングの許可」（App Tracking Transparency）の 確認が表示されます。許可しない場合でも本アプリはご利用でき、広告はパーソナライズされない形で表示されます。
  • EU/EEA・英国などの地域：適用される法令（GDPR等）に基づき、広告に関する同意取得メッセージ （Google の User Messaging Platform）が表示されます。同意の内容はいつでも変更できます。
  • 利用者は、デバイスの設定からいつでもパーソナライズド広告をオプトアウトできます（オプトアウト後も広告は 表示されますが、興味に基づかない一般的な広告となります）。
    • iOS：設定 → プライバシーとセキュリティ → トラッキング
    • Android：設定 → Google → 広告

7．データの保存期間

  • 本アプリ自体は、利用者の情報を当方のサーバーに保存しません（当方は本アプリ用の専用サーバーを持ちません）。
  • 広告・マルチプレイに関する情報の保存期間は、各第三者サービス（Google／Unity）のポリシーに従います。

8．児童の安全について（児童の性的虐待と搾取（CSAE）に対する基準）

当方は、児童の安全と保護を最優先事項として取り組んでいます。本アプリは、児童の性的虐待と搾取（CSAE）に 関するあらゆるコンテンツや行為を明示的に禁止します。

  • 本アプリは13歳未満の利用者を主な対象とせず、利用者が投稿・共有を行う機能を持ちません。
  • 本アプリは、米国 COPPA・EU GDPR 等の児童保護法令に配慮し、広告を非パーソナライズ（興味・関心に 基づかない一般的な広告）に限定して配信します。これにより、13歳未満の利用者に対する行動ターゲティング 広告は行いません。
  • 当方は、13歳未満の子どもから意図的に個人情報を取得することはありません。万一、保護者の同意なく13歳未満の 子どもの情報が取得されたことが判明した場合、当方は速やかに当該情報を削除するよう努めます。
  • CSAE に関する懸念事項や通報については、第1項のお問い合わせ窓口までご連絡ください。法的に必要な場合、 関連当局への通報を行います。

9．カリフォルニア州にお住まいの方の権利（CCPA／CPRA）

カリフォルニア州消費者プライバシー法（CCPA／CPRA）に基づき、カリフォルニア州にお住まいの方には以下の 権利が認められています。本アプリが取り扱う情報は、第2項に記載の広告ID・端末情報・匿名認証IDに限られます。

  • 知る権利：当方が取得する情報のカテゴリ・利用目的・提供先について知る権利。
  • 削除を求める権利：当方が保有する情報の削除を請求する権利（法令上保持が必要な場合を除く）。
  • 販売・共有のオプトアウト権：広告目的での情報の「販売」または「共有」を停止するよう求める権利。
  • 権利行使による差別を受けない権利：これらの権利を行使したことを理由に、不利益な取り扱いを受けません。

当方は情報を第三者に販売しません。広告目的での情報利用（パーソナライズド広告）は、本アプリの設定画面の 広告同意の再設定、または端末のトラッキング設定からいつでもオプトアウトできます（第6項参照）。その他の ご請求は、第1項のお問い合わせ窓口までご連絡ください。

10．情報の開示・訂正・削除

利用者から情報の開示・訂正・削除等のご請求があった場合、本人確認の上、法令の定めに従って対応します。 これらのご請求は、第1項のお問い合わせ窓口までご連絡ください。

11．本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本アプリ内または本 ページ上で告知します。変更後に本アプリをご利用された場合、変更後のポリシーに同意したものとみなします。
